Django Template Example Showcase

1. Display Variables

Course Name: {{ name }}

Duration: {{ duration }}

Total Seats: {{ seats }}

2. Conditional Rendering (if/else)

{% if name %}

Hello, the course name is available.

{% else %}

No course name provided.

{% endif %}

3. Looping Through List

    {% for student in students %}
  • {{ forloop.counter }}. {{ student }}
    • {% empty %}
  • No students found.
    • {% endfor %}

4. Looping Through Dictionary

    {% for key, value in course_info.items %}
  • {{ key|capfirst }}: {{ value }}
    • {% endfor %}

5. Template Filters

Uppercase: {{ name|upper }}

Lowercase: {{ name|lower }}

First letter capital: {{ name|capfirst }}

Word Truncate (3): {{ duration|truncatewords:3 }}

Char Truncate (10): {{ duration|truncatechars:10 }}

Slice (first 6 chars): {{ name|slice:":6" }}

Length of name: {{ name|length }}

6. Default Filter (Fallback Value)

Empty var: {{ empty_var|default:"No value provided" }}

None var: {{ none_var|default_if_none:"Null value replaced" }}

7. Float Format

p1 (3 decimal): {{ p1|floatformat:3 }}

p2 (0 decimal): {{ p2|floatformat:0 }}

p3 (default): {{ p3|floatformat }}

p3 (remove trailing zero): {{ p3|floatformat:-2 }}

8. Date and Time Formatting

Raw datetime: {{ dt }}

Date only: {{ dt|date:"Y-m-d" }}

Time only: {{ dt|time:"H:i:s" }}

Full datetime formatted: {{ dt|date:"D d M Y H:i" }}

9. Safe Filter (Render HTML inside variable)

Without safe: {{ html_content }}

With safe: {{ html_content|safe }}

10. Escaping

{% autoescape on %}{{ html_content }}{% endautoescape %}

{% autoescape off %}{{ html_content }}{% endautoescape %}
